Jeremy Daugherty

AI puzzle 3 program.

Requires either a command-line declaration of what file it's using to pull it's data
or uses a file called puzzle.txt. If either is not of the format given on the website
then the program is likely to break.

Format

(Side length)  (Number of colors)
grid data in row/column format

Running the Program
To run the program simply run the P3_Daugherty_j.py file in python 3.4 or 3.5.

Output
The Program outputs the results to both the console and a file called solution.txt.
